# Frontend Design Skill

name: frontend-design
description: Create distinctive, production-grade frontend interfaces with high design quality. Use this skill when the user asks to build web components, pages, artifacts, posters, or applications (examples include websites, landing pages, dashboards, React components, HTML/CSS layouts, or when styling/beautifying any web UI). Generates creative, polished code and UI design that avoids generic AI aesthetics.

## Purpose
This skill guides the creation of distinctive, production-ready frontend interfaces. It is used when the user wants a visually striking and usable design system, component, page, or interaction.

## When to use
- Building or redesigning UI for a web page, dashboard, modal, component, or marketing page
- Implementing a visual concept with strong brand, layout, typography, or motion requirements
- Creating polished React/Vue/HTML/CSS interfaces that need a creative direction

## Behavior
1. Start by clarifying the interface purpose, audience, and technical constraints.
2. Commit to a strong aesthetic direction with a clear concept.
3. Apply bold but intentional typography, color, spacing, and composition.
4. Prefer unique visual choices over generic defaults.
5. Implement real working frontend code that is maintainable and production-quality.

## Quality criteria
- Distinctive aesthetic point-of-view
- Cohesive and polished visual system
- Production-grade implementation with clean structure
- Attention to spacing, motion, typography, and interaction details
- Avoids overused generic UI patterns and bland themes

## Example prompts
- "Design a premium event landing page with a cinematic hero and action cards"
- "Build a responsive ticket checkout modal with refined spacing and motion"
- "Create a new React promo hero component with bold typography and layered visuals"
